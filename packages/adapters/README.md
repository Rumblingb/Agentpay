# packages/adapters — Adapter landing area

This package directory is a planned home for third-party and first-party
adapter implementations. It contains adapter interface specs and minimal
examples. Adapter implementations are added in follow-up PRs.

See `docs/adapters/MINIMAL_ADAPTER_INTERFACE.md` and
`docs/adapters/POST_PR7_ADAPTER_IMPLEMENTATION_PLAN.md` for details.
